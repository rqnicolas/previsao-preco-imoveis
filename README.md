# Previsão de Preço de Imóveis

Este é um projeto simples de Machine Learning desenvolvido com o objetivo de aprendizado prático.  
Aqui aplico conceitos básicos de regressão linear para prever o preço de imóveis a partir de algumas características.

O foco do projeto é entender, na prática, como funciona o fluxo de um modelo de Machine Learning, desde a leitura dos dados até a avaliação dos resultados.

## O que foi utilizado
- Python
- pandas
- scikit-learn

## Como o projeto funciona
- Leitura de dados a partir de um arquivo CSV
- Separação das variáveis de entrada (tamanho, quartos e banheiros) e da variável de saída (preço)
- Divisão dos dados em treino e teste
- Treinamento de um modelo de Regressão Linear
- Geração de previsões
- Avaliação do modelo utilizando Erro Médio Absoluto (MAE)

## Objetivo
Este projeto foi criado com fins educacionais, como parte do meu processo de aprendizado em Machine Learning e análise de dados.  
A ideia é evoluir o projeto conforme avanço nos estudos.
